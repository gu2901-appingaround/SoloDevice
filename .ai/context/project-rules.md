# Project Rules

- TypeScript estrito, sem any
- Dominio em src/modules/*
- UI em src/core/ui e src/app
- Copia centralizada em src/core/copy
- Git obrigatorio desde o primeiro commit
- Repositorio remoto deve ser criado antes da entrega
- Codigo inicial deve ser enviado para o remoto
- Na primeira interacao no workspace, executar checklist do agente Git Specialist
- Se a sessao ainda nao estiver no workspace do projeto, orientar abertura da pasta e executar o checklist na primeira interacao apos abrir
