# Git Setup

Este projeto deve ser versionado com Git desde o inicio. Antes de considerar a entrega concluida, garanta que o repositorio remoto foi criado e que o codigo inicial foi publicado.

## O que ja e automatico no scaffold
1. Inicializacao do repositorio local
2. Definicao da branch main
3. Criacao da branch test
4. Configuracao local de user.name e user.email
5. Commit inicial com mensagem "chore: initial scaffold"

## Passos obrigatorios restantes
1. Crie o repositorio remoto no provedor Git da equipe.
2. Configure o remoto:
   git remote add origin <repo-url>
3. Publique o projeto:
   git push -u origin main

## Criterio de concluido
- Git local inicializado e configurado
- Branches main e test existentes
- Commit inicial criado automaticamente
- Repositorio remoto criado
- origin configurado
- Push para a branch principal executado com sucesso
