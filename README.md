# meusuperproject1010101

Projeto inicial criado pelo SoloDevice Project Generator.

## Especialidade
Projeto front-only com deploy em Vercel

## Stack alvo
Front: React + TypeScript | Back: nao aplicavel | Banco: sem base de dados | Auth: simple

## Objetivo do MVP
10 fotos de pistas de corrida scrolando na tela

## Como rodar
1. npm install
2. npm run dev
3. npm run lint
4. npm run typecheck

## Fluxo Git
O scaffold ja cria e configura automaticamente o repositorio local com:
- branch main
- branch test
- commit inicial (chore: initial scaffold)

Passos restantes para publicacao:
1. Criar o repositorio remoto no provedor Git adotado pelo time
2. Configurar origin com git remote add origin <repo-url>
3. Publicar com git push -u origin main

Considere o projeto pronto apenas quando as branches main e test existirem, o repositorio remoto existir e o codigo inicial estiver publicado nele. Veja tambem o arquivo GIT_SETUP.md.
